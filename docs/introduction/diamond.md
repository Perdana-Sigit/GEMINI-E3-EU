# GEMINI-E3-EU within the DIAMOND Project

The current version of GEMINI-E3-EU has been developed and upgraded under Work Package 3 (WP3) of the DIAMOND project (“Delivering the next generation of open Integrated Assessment Models for Net-zero, sustainable Development”).

The objective of the upgrade is to improve the empirical consistency, geographical granularity, sectoral representation, and policy relevance of the model while ensuring compatibility with the harmonised scenario framework used across DIAMOND Integrated Assessment Models (IAMs).

The upgraded version strengthens the capacity of GEMINI-E3-EU to support:

-	European climate policy assessment
- Integrated climate-energy-economic analysis
- Distributional and labour market analysis
- Regional decarbonisation studies
- Air pollution and environmental co-benefit assessment
- Comparative multi-model exercises within DIAMOND.

The model also supports future integration with external modules and regional frameworks developed within the project.
